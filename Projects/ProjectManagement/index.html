Project Management Plan for a Vertical Farming Enterprise

Project Summary

EcoElevate was executed as a collaborative project by a team of six members, focusing on developing a comprehensive project management plan for a solar-powered vertical farming system.

This project involved developing a comprehensive Project Management Plan (PMP) for EcoElevate, a modular solar-powered vertical farming system. The plan follows a structured systems engineering approach. It includes detailed planning for project organization and staffing, defining team structures, roles, responsibilities, and communication flows. The PMP also incorporates a work breakdown structure, schedule, network scheduling, milestones, and entrance/exit criteria to ensure project deliverables are completed efficiently. Cost management is addressed through labor rates, cost breakdowns, time-phased budgets, and earned value management planning. Finally, the plan includes a comprehensive risk management framework, identifying potential risks, their likelihood and severity, and mitigation strategies. This PMP demonstrates a systematic, multi-disciplinary approach to planning, coordinating, and executing a complex project.

My Contribution

  1. Project Scope
    • Defined the overall project scope, clearly outlining objectives, boundaries, and deliverables.
    • Developed the Context Diagram to illustrate system interactions, and external dependencies.

      • Shows the Vertical Farming System as the system of interest and distinguishes controllable farm operations from external entities such as suppliers, operators, customers, regulators, and the power utility. The diagram highlights the scope of the project and how the system interacts with the external entities to achieve system goals.

  2. Staffing
    • Defined the Linear responsibility chart based off of staffing data to ensure scope of responsibility of each department for given task.

      • Maps project tasks to roles and departments, clearly defining responsibility, accountability, consultation, and information flow across all project phases. The diagram clarifies task ownership, decision authority, and communication structure, supporting coordinated execution and effective oversight throughout the project lifecycle.

  3. Work Breakdown Structure and Schedule
    • Created the Gantt Chart, mapping tasks, durations, and dependencies.
    • Assigned appropriate contributors to each task, ensuring accountability and efficient workflow.

      • Shows the EcoElevate project schedule created in ProjectLibre, with tasks assigned to project roles using acronyms to represent functional responsibilities. The chart illustrates task sequencing, durations, and role-based resource allocation across the project timeline.

  4. Network Scheduling
    • Developed the project schedule with time allocations for tasks and key milestones.
    • Assessed schedule uncertainty using three-point estimation and sigma variation analysis.

      • Evaluates schedule uncertainty using three-point estimation by calculating expected durations, standard deviation (sigma), and variance for critical path activities. The analysis defines sigma ranges for potential early or delayed completion, supporting risk-aware and informed project planning. (Detailed estimation variation table is present in Appendix)

  5. Cost Management
    • Prepared a time-phased budget, distributing estimated labor and material costs across the project timeline.
    • Defined expected project profit and management reserves to address uncertainties.

      • Shows cumulative project costs over biweekly periods, including planned budget, management reserve, and expected profit. The chart enables clear comparison of planned spending, contingency allocation, and projected return across the project lifecycle.

  6. Risk Management
    • Developed the complete Risk Management Plan, assigning stakeholder responsibilities, identifying and classifying risks, and analyzing mitigation strategies for critical risks.

      • Shows identified project risks along with their likelihood and impact, and visualizes risk prioritization using the risk matrix. This supports clear identification of critical risks and informs focused risk response planning.

    Tool Used: Project Libre, Excel
